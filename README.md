# Flash Telegram Bot

A Telegram bot to control your PC remotely.

## Features
- PC Status monitoring
- Process management
- Screenshot capture
- Remote shutdown
- Multi-monitor support

## Setup
1. Create a Telegram bot using BotFather
2. Copy your bot token
3. Run setup_gui.py
4. Enter your bot token and user ID
5. Click Build Bot
